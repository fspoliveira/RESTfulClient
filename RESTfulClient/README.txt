RESTfulClient é um programa desenvolvido em RESTful com o Framework RESTlet.
O programa se conecta no servidor da Agenda de Contatos que está no projeto RESTfulSever.
Para efetuar as operações com o servidor é criado uma string no formado JSON à partir da classe Bean.

OPERAÇÕES SUPORTADAS PELA AGENDA DE CONTATOS
============================================================================================================================

a) Inserir um contato
b) Excluir um contato
c) Consultar um contato pelo e-mail
d) Listar todos os contatos
